Method Documentation

- (BOOL) updateWithXML: (OOTBXMLElement *)  xml

Assumptions: 1) There can be multiple wrappers (we set no limit on the redirects). 2) If the wrapper has multiple linear creatives with sequence numbers AND the child has multiple linear creatives with corresponding sequence numbers, we put the right tracking events to the correct linear creatives of the child. 3) If the wrapper has multiple linear creatives but the child's linear creatives do not have sequence numbers we assume that the order that the creatives are given is the order that we want to match with the wrapper's sequence numbers
